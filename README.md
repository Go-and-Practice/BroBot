<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* python
```sh
sudo apt update
sudo apt install python3 python3-pip -y
```

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/csivitu/BroBot.git
cd BroBot
```
2. Install pip packages
```sh
pip3 install -r requirements.txt
```
